Sauvegarde & Synchro

Vos données sont stockées localement sur votre iPhone/iPad dans une base de données sécurisée. Exportez ou importez vos configurations à tout moment.

Synchronisation iCloud (Manuelle)

Vous pouvez partager vos données entre tous vos appareils Apple (iPhone, iPad, Mac) via votre compte iCloud personnel :

  • Pour sauvegarder : Cliquez sur Exporter, puis dans l'écran de partage Apple, choisissez Enregistrer dans Fichiers et placez le fichier dans un dossier sur votre iCloud Drive.
  • Pour synchroniser : Sur vos autres appareils Apple, cliquez sur Importer, ouvrez votre iCloud Drive et sélectionnez ce même fichier.
💡 Astuce : Si vous écrasez le même fichier à chaque export, iCloud se charge de répercuter instantanément la version la plus récente sur tous vos appareils !

À propos de FPV Manager

Conçu pour les pilotes exigeants. Permet un suivi précis de vos cycles LiPo/LiIon, stocke vos factures pour les assurances en cas de perte de drone, et permet de configurer vos builds méthodiquement.
